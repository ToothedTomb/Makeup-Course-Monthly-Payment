# Makeup Course Monthly Payment
# About:
We are a makeup school that teaches women to learn makeup. The course is $100 but monthly, you pay $10.
Only the admin can see all the users data, the user for example Ella can only see her data and no one else. So each user can sign in and see how much they need to pay back and the set deadline payment. So it has a login and logout feature. This was done to help me learn Django. :)
